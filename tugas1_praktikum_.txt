3. jelaskan apa yang di maksud dengan perintah git,init,clone,remote,add,commit,push dan checkout ?
 >git init perintah ini di gunakan untuk menginisialisasi repositori git baru di local direktori 
 >clone di gunakan untuk menduplikasi yang ada baik lokal ataupun remot,ke dalam komputer lokal 
 >remote perintah ini di gunakan untuk mengola referensi remote dalam repositorigit.
 >add perintah untuk menambahkan perubahan yang sudah di lakukan pada file (baik itu file baru atau perubahan pada file yang sudah ada ) ke dalam staging area.
 >push perintah yang di gunakan untuk mengirim commit yang ada di repositori remote,seperti github atau git lab.
 >checkout di gunakan untuk berpindah antar cabang (branch) atau mengembalikan file ke kondisi tertentu.

5.apa bedanya variable user dengan variable system ?
 variable user di gunakan untuk menyimpan data atau pengaturan yang bergubungan dengan pengguna tertentum
 variable system berhubungan dengan pengaturan dan data yang di perlukan oleh sistem operasi atau aplikasi yang berjalan di seluruh sistem
